How to use CropCanvas

Set your canvas size using the W and H inputs in the toolbar, or pick a preset. Then add images, text, or shapes to your canvas.

Tools

  • V Select - click to select, drag to move, corner handles to resize
  • T Text - adds editable text. Double-click to edit inline
  • I Image - opens file picker. You can also drag-drop images onto the canvas
  • R Rectangle - adds a rectangle with fill, stroke, and corner radius
  • C Circle - adds a circle or ellipse
  • L Line - adds a straight line
  • A Arrow - adds a line with an arrowhead

Zoom & Navigation

  • Ctrl+Scroll Zoom in and out with mouse wheel
  • Zoom out - decrease zoom level by 10%
  • Zoom in - increase zoom level by 10%
  • 100% Reset zoom - click the percentage to go back to 100%
  • Fit to screen - auto-zoom to fill the available workspace

The rulers show 0 at the center of the canvas and count outward in both directions.

Properties Panel

When you select an element, the Properties tab shows editable values:

  • Transform - X, Y position, Width, Height, Rotation, Opacity
  • Typography (text only) - font, size, weight, color, bold/italic/underline, alignment
  • Shape (shapes only) - fill color, stroke color, stroke width, corner radius

Changes apply instantly as you type or pick colors.

Layers Panel

The Layers tab lists all elements on the canvas.

  • Visibility toggle - click the eye icon to show/hide a layer. Hidden layers are not exported.

Click any layer name to select it on the canvas.

Canvas Settings

At the bottom of the Properties tab, expand "Canvas Settings" to:

  • Change the canvas background color
  • Toggle the grid overlay on or off

Keyboard Shortcuts

  • Ctrl+Z Undo / Ctrl+Shift+Z Redo
  • Ctrl+S - save
  • Delete or Backspace - delete selected element
  • Arrow keys - nudge 1px (Shift + arrow = 10px)
  • Ctrl+Scroll - zoom

Export

Switch to the Export tab in the right panel. Choose PNG or JPEG, set a filename, and click Download. Only content inside the canvas viewport is exported. Overflow content is excluded.

Tips

  • Your work is auto-saved. Refresh the page and your canvas will be restored.
  • Save button (top right) or Ctrl+S to save manually.
  • Reset button (top right) to clear everything and start over.
  • The sun/moon icon at the bottom of the tool rail toggles dark and light theme.
